Autograph Sound

Status: in development

Autograph Sound supply high-spec audio equipment to the events and entertainment industries. The company already had a website, built some time ago using Flash. Ahead of a future, full-scale redesign they wanted the existing site rebuilt using current technology.

Part of the brief was that the original design had to be replicated as closely as possible. Below are some some side-by-side screenshots.

current Flash site
new site

Using the same colour scheme as the existing website, I also built a simple, responsive mobile site, as well as supplying an updated, vectorised version of the company logo that would scale nicely for future projects / branding.

The multitalented Chris, Autograph’s unofficial in-house developer, had already started work on the backend code for the new site when I was brought in to help. I liaised with Chris throughout the project to make sure the different parts of the “stack” all tied together.

Technical highlights:

  • Backend + CMS: Rails, CaseIn
  • Frontend: Hybrid clientside routing using PageJS library, lots of custom CSS + JS to replicate existing styling + animations.
